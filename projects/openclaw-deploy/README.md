# OpenClaw Deploy - 自动化部署平台

> 一键部署你的 AI 助手

## 项目结构

```
openclaw-deploy/
├── frontend/          # 前端页面 (Vue3)
├── backend/           # 后端服务 (Spring Boot)
├── scripts/           # 部署脚本
└── docs/              # 文档
```

## 快速开始

```bash
# 前端
cd frontend && npm install && npm run dev

# 后端
cd backend && mvn spring-boot:run
```

## 功能特性

- ✅ 一键部署 OpenClaw
- ✅ 可视化配置管理
- ✅ 实时状态监控
- ✅ Skills 快速安装
- ✅ 多渠道通知配置
